# AnimalsCS

## Описание

Проект "AnimalsCS" демонстрирует основные концепции объектно-ориентированного программирования, такие как абстрактные классы, интерфейсы, наследование и полиморфизм. Проект включает в себя систему управления животными, которая позволяет создавать животных, добавлять их в приют и управлять ими.

## Компоненты системы

1. **Абстрактный класс `Animal`**:
   - Содержит абстрактный метод `MakeSound()` и виртуальный метод `Move()`.
   - Содержит свойства `Id`, `Name`, `IsOn` и `Type`.

2. **Интерфейс `IPet`**:
   - Содержит метод `Play()` и свойство `Name`.

3. **Классы `Dog` и `Cat`**:
   - Наследуют `Animal` и реализуют `IPet`.
   - Содержат дополнительные свойства и методы для управления животными.

4. **Класс `Shelter`**:
   - Управляет списком животных.
   - Содержит методы для добавления и удаления животных, а также для вызова методов `MakeSound()` и `Move()` для всех животных.

5. **Класс `Program`**:
   - Демонстрирует функциональность системы, создавая животных, добавляя их в приют и управляя ими.

## Как запустить программу

1. Откройте командную строку (Command Prompt) и перейдите в директорию проекта:
   ```
   cd AnimalsCS
   ```

2. Скомпилируйте проект с помощью команды:
   ```
   dotnet build
   ```

3. Запустите программу с помощью команды:
   ```
   dotnet run
   ```

## Описание работы программы

Программа создает приют и добавляет в него двух животных: собаку и кошку. Программа вызывает методы `MakeSound()` и `Move()` для всех животных в приюте. Программа также обрабатывает некорректные входные данные, бросая исключение `ArgumentException` при попытке создать животное с пустым именем.
